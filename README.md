## 🎙️ Featured Project — VoiceForge

> **A production-grade AI voice platform** built to compete with ElevenLabs — multi-language TTS, voice cloning, persistent history, full auth. Deployed. Real users. $1.90/month infrastructure.

**[voiceforge-gamma.vercel.app](https://voiceforge-gamma.vercel.app)** · **[GitHub Repo](https://github.com/Smruthi-Sri-Datta/Voiceforge)**

```
React + Vite  →  FastAPI (Render)  →  RunPod Serverless GPU  →  Supabase
     ↑                  ↑                      ↑                    ↑
  Frontend           Auth + DB            XTTS v2 + Sarvam      Storage + PostgreSQL
                     Webhooks             Voice Cloning          Persistent History
```

**What makes it production-grade:**
- ⚡ **Async webhook architecture** — solves Render's 30s timeout for 43–90s GPU inference jobs
- 🌍 **16 languages, 2 engines** — XTTS v2 for global languages, Sarvam AI for Indian languages
- 🎙️ **Voice cloning** — upload a 30s WAV, speak in that voice forever
- 🔐 **Full auth** — Google OAuth + Email/OTP + forgot-password flow
- 💰 **$1.90/month** — serverless GPU, free-tier backend/frontend/DB
